바이트 단위 분할

UTF-8 기준으로 지정한 바이트 길이마다 텍스트를 나눕니다. 멀티바이트 문자도 안전하게 처리됩니다.

어떻게 쓰나요?

1) 텍스트 붙여넣기 → 2) 바이트 길이 입력 → 3) 실행 (단축키: Ctrl/⌘ + Enter)

FAQ

바이트는 어떻게 계산하나요?

UTF-8 인코딩 기준으로 계산합니다.

멀티바이트 문자를 잘라먹지 않나요?

문자 경계를 계산해 안전하게 분할합니다.

구분자나 줄바꿈을 바꿀 수 있나요?

옵션에서 구분자와 줄바꿈 개수를 지정할 수 있습니다.

결과를 저장할 수 있나요?

복사하거나 텍스트 파일로 다운로드할 수 있습니다.
